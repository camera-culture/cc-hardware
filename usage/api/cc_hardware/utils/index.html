class Manager(*, cleanup_on_keyboard_interrupt=True, **components)#

This is a manager for handling components which must be closed. It is essentially just a context manager which calls close on all components when it is closed.

run(iter=0, *, setup=None, loop=None, cleanup=None)#

Runs a setup and loop function until all components are okay.

Parameters:

iter (int, optional) – Iteration counter. Defaults to 0.

Keyword Arguments:

  • setup (Callable[..., bool | None] | None, optional) – Setup function to run before the loop. Accepts keyword arguments, returns None. Defaults to None. A return value of False will stop the run.

  • loop (Callable[..., bool | None] | None, optional) – Loop function to run until all components are okay. Accepts keyword arguments, returns bool. When False, the loop will stop and cleanup will begin. Defaults to None. A return value of None will continue the loop.

  • cleanup (Callable[..., None] | None, optional) – Cleanup function to run after the loop. Accepts keyword arguments, returns None. Defaults to None.

class Registry#

Base class providing a registry for its subclasses, plus a factory method.

This class supports both direct and lazy registration of subclasses and can also associate a registered class with a ‘friend’ (another class not necessarily inheriting from Registry).

Examples

Register directly:

@MyRegistry.register class Foo(MyRegistry):

pass

Register by name (lazy):

MyRegistry.register(“Bar”, “my_module.submodule”)

Create instance:

instance = MyRegistry.create_from_registry(“Foo”)

Create instance from one of many registries:

instance = MyRegistry.create_from_registry(“Bar”)
