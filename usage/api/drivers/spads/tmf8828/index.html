drivers.spads.tmf8828#

TMF8828 sensor driver for SPAD sensors.

The TMF8828 is a 8x8 multi-zone time-of-flight sensor made by AMS. It uses a wide VCSEL and supports custom mapping of SPAD pixels to allow for 3x3, 4x4, 3x6, and 8x8 multizone output. The TMF8828Sensor class was developed to interface with the TMF882X Arduino Shield.
