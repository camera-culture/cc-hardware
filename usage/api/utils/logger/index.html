utils.logger#

Logging utilities for the cc_hardware package.

This module provides a custom logger that uses tqdm.write to log messages to the console. It also provides a filter to set a maximum logging level for a logger.

Example:

from cc_hardware.utils.logger import get_logger

get_logger().info("This is an info message.")

emit(record)[source]#

Emit a record.

If a formatter is specified, it is used to format the record. The record is then written to the stream with a trailing newline. If exception information is present, it is formatted using traceback.print_exception and appended to the stream. If the stream has an ‘encoding’ attribute, it is used to determine how to do the output to the stream.
